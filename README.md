# sora2video
소라2 생성 비디오


```
장면 의도: TPS 게임용 짧은 보이스 컷 시퀀스. 애니메이션풍, 12세 소녀 '미야코'이 전투 상황에서 짧게 반응하는 음성·표정 컷들을 연속으로 생성. 모든 연출은 비성적, 교복 착용.

스타일/톤: 깔끔한 애니 성우톤, 아이같은 또렷함과 약간의 긴장감. 한국어 표준 억양. 각 컷은 1~3초 내외로 짧게.

카메라/프레이밍 공통: 상반신 중심(허리 위), 표정이 잘 보이도록 약간의 클로즈업. 배경은 흐릿한 전장(연기·파편 디테일 약하게), 조명은 자연광. 불필요한 신체 노출 금지.

[컷1] "흥"
- 표정: 뾰로통, 콧방귀.
- 음성: [미야코: 12세, 밝은 톤, 짧고 단호] "흥!"
- 사운드 큐: 배경 무음, 아주 짧은 코맹맹이 톤 강조.
- 길이: 1s

[컷2] "쳇"
- 표정: 불만 섞인 눈썹, 입술 삐침.
- 음성: [미야코: 12세, 짜증 섞인 톤] "쳇…!"
- 사운드 큐: 발끝 소리(짧음) 또는 금속성 경고음(아주 약함).
- 길이: 1.2s

[컷3] "꺅"
- 표정: 깜짝 놀란 눈, 손으로 얼굴 옆을 가볍게 막는 동작(과하지 않게).
- 음성: [미야코: 12세, 높은 음역의 놀람] "꺅!!"
- 사운드 큐: 짧은 스틸 충격음(하이헷 같은 짧음).
- 길이: 0.8–1s

[컷4] "으아아"
- 표정: 과장된 고통/놀람(비폭력적), 몸이 휘청하는 모션.
- 음성: [미야코: 12세, 길게 절규 비슷하지만 과장되지 않게] "으아아아!!"
- 사운드 큐: 바닥에 가볍게 부딪히는 둔탁음(무해한 느낌).
- 카메라: 클로즈업 약간 흔들림.
- 길이: 1.5–2s

[컷5] "수류탄 투척"
- 상황: 수류탄을 던지려는 행동. 전투 지시형 음성.
- 표정/동작: 집중한 표정, 오른손(또는 왼손) 던지기 동작(과장X).
- 음성: [미야코: 12세, 결의에 찬 낮은 톤(아이치곤 약간 단단하게)] "수류탄 투척!"
- 사운드 큐: 짧은 윈드업(던지기 전 소리) + 휙 하는 공기음, 던진 뒤 아주 짧은 딸깍(핀 뽑는 소리) 가능.
- 길이: 1.2s

[컷6] "탄약이 필요해"
- 상황: 아군에 통신/요청을 보내는 짧은 라인.
- 표정: 걱정 섞인 차분한 표정, 손으로 총의 탄창을 가리키는 제스처(과장X).
- 음성: [미야코: 12세, 약간 낮고 긴장된 톤, 정중한 호소] "탄약이 필요해!"
- 사운드 큐: 무전기 잡음(아주 약함) 또는 짧은 비프음.
- 길이: 1.5s

[컷7] "엄호해줘"
- 상황: 아군에게 엄호 요청.
- 표정: 단호하고 침착한 표정, 눈은 전방 응시.
- 음성: [미야코: 12세, 명확하고 빠른 톤, 지휘형] "엄호해줘!"
- 사운드 큐: 배경에서 먼지 소리나 총성 희미하게(낮게), 요청 직후 짧은 무전 전송음.
- 길이: 1.0–1.3s

출력 포맷: 각 컷은 개별 클립(별도 파일)로도, 시퀀스(연속 컷)로도 생성 가능. 각 컷에 대해 '음성 우선'을 표기했으므로 입모양이 대사에 최대한 맞게 생성되도록 타이밍을 명시. 모든 연출은 비성적이며 교복은 단정하게 표현.
```
